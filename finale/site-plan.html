Desert Rain Soap Site Plan

Chelsea Saunders

WDD 131-A1

Overview

Purpose

To create a personal and functional website that serves as a practical tool for soap-making enthusiasts, allowing them to calculate and customize soap recipes with ease while also showcasing and organizing my favorite soap recipes for personal reference and sharing with friends, family, or the broader community.

Audience

Soap-making enthusiasts and hobbyists who are interested in creating and customizing their own soap recipes. This includes individuals who enjoy crafting natural, handmade products for personal use, as well as friends, family, or anyone looking for inspiration or tools to simplify the soap-making process.

Dynamic elements

* For the Soap calculator, I will need a dynamic interactive input form which will allow users to enter ingredient amounts and superfat percentages
* A "calculate" button which will trigger javascript to process the input and display the results dynamically.
* Input validation, conditional branching to show an alert if the user enters invalid data and a live display of calculation results
* For the recipes page, use an array of recipe objects to store information (recipe name, Image and description), dynamically create and display recipe cards on the page using javascript use forEach or map to iterate over the recipe array and generate content.
* Allow sorting or filtering recipes based on criteria like ingredients or soap type.
* Maybe a toggle button to switch between light and dark mode, dynamically updating the site's style.
* Clicking on a recipe image opens a larger version in a modal window.
* Dynamic JavaScript to handle the image display and close button.
* Add dropdowns or buttons for sorting recipes (e.g., alphabetical order, by ingredient type).
* Use filter or sort methods on the recipe array to dynamically update the displayed recipes.
